Privacy Policy

Glossary GPT is a custom AI assistant that reads and updates a bilingual glossary stored at Glossario.csv in the GitHub repository Cristale92/Glossary-API.

What data we process

  • User prompts and instructions provided to the GPT inside OpenAI’s interface.
  • Repository details necessary to operate (owner/repo, file path Glossario.csv, branch main).
  • File contents of Glossario.csv to read and commit updates.
  • Commit metadata (commit message, file SHA, timestamps) when changes are made.

Authentication & tokens

Access to GitHub is performed using a Personal Access Token (PAT) that you configure within OpenAI Actions. The token is transmitted by OpenAI to GitHub for the API calls and is not collected by us outside of OpenAI’s platform. You may revoke it anytime in your GitHub settings.

How we use data

Data is used solely to maintain your glossary (read, add, update, delete entries) as requested by you. We do not sell your data. Data is shared only with GitHub via its REST API in order to read/write the file.

Data retention

  • No separate database is maintained by us; your glossary persists in your GitHub repository and commit history.
  • Conversation data may be retained by OpenAI per their policies.

Your choices

  • Revoke or rotate your GitHub PAT at any time.
  • Edit or delete the glossary file in your repository.
  • Delete commits or the repository according to GitHub’s features.

Third parties

This GPT interacts with GitHub’s REST API. Please review GitHub’s own privacy practices.

Children’s data

This service is not intended for children under the age required by local law. We do not knowingly collect such data.

Changes to this policy

We may update this Privacy Policy from time to time. Material changes will be reflected by updating the effective date above.
