Privacy Policy for Glossary GPT

Effective date: December 4, 2025

Who we are

Glossary GPT is a custom AI assistant that maintains a bilingual glossary by editing a CSV file in the GitHub repository Cristale92/Glossary-API.

What data we process

  • User prompts and instructions sent to the GPT.
  • Repository metadata: owner/repo name, file path (Glossario.csv), branch name (main).
  • File contents of Glossario.csv for the purpose of reading and committing updates.
  • Commit metadata (commit message, SHA).

Authentication and tokens

GitHub access uses a Personal Access Token (PAT) you configure inside OpenAI’s Actions. The token is handled by OpenAI’s integrations and is not collected by us outside of the OpenAI platform. You can revoke access by removing the token or revoking it in GitHub settings.

How we use data

Data is used solely to read and update the glossary CSV as requested by the user. We do not sell or share data with third parties, other than processing it via GitHub’s API to read/write the file.

Data retention

No additional persistent storage is used by us beyond your GitHub repository. Conversation data may be retained by OpenAI per their policies. Commits remain in your GitHub repo history.

Your choices

  • Revoke the GitHub PAT at any time.
  • Delete or modify the glossary file and commits in your repository.

Third parties

This GPT interacts with GitHub’s REST API. Please refer to GitHub’s privacy practices as well.
